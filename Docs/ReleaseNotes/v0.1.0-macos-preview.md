# Screen Q macOS Preview 0.1.0

This is the first public macOS preview for Screen Q.

## Included

- Notarized Developer ID macOS app.
- Local-first Screen Q Native host and viewer surfaces.
- Explicit pairing-code flow and host approval gate.
- Per-session viewer permission controls.
- Mac Screen Sharing, Generic VNC, and RDP connection surfaces.
- RDP certificate review, pinning, and Keychain-backed credential storage.
- iCloud key-value sync kept available for future iOS and iPadOS work.
- Public release guard CI for unfinished extension, platform, signing, and
  debug-only feature regressions.

## Not Included

- Mac App Store distribution.
- Public-internet relay service.
- Third-party remote control of iPhone or iPad.
- Publicly embedded ReplayKit Broadcast Upload Extension.
- Production iOS TestFlight release.

## Verification

- macOS Developer ID archive signed successfully.
- Apple notarization accepted.
- Stapling validated.
- Gatekeeper accepted the app as Notarized Developer ID.
- Public-release guard passed against source and built artifacts.

## Install Notes

Download the macOS preview zip, unzip it, and move `Screen Q.app` to
`/Applications`. On first run, grant Screen Recording, Accessibility, and Local
Network permissions when using this Mac as a host.

For source builds, keep the default `com.example.Screen-Q` bundle identifier or
override `SCREENQ_BUNDLE_ID` with your own registered bundle identifier.
